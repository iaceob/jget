下载终端
此项非常重要, 这个名称必须和下载终端中的 name 相同, 如果下载终端中没有配置 name 则默认提取下载终端的机器名称
此项可不用更改, ip 会在下载机的心跳中进行更新

id: ${clis[i].id}

name: ${clis[i].name}

ip: ${clis[i].ip}

heartbeat: ${clis[i].heartbeat, dateFormat='yyyy-MM-dd HH:mm:ss'}

expired: ${clis[i].expired}

ctime: ${clis[i].ctime, dateFormat='yyyy-MM-dd HH:mm:ss'}
